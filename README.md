# Hotel Booking Demand Analysis

## Project Overview
This project analyzes hotel booking data to understand booking patterns, cancellations, customer behavior, and demand trends.

## Objectives
- Analyze booking and cancellation behavior
- Identify factors affecting cancellations
- Explore seasonality and customer segments
- Perform exploratory data analysis (EDA)

## Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values and outliers
- Categorical and numerical feature analysis
- Cancellation trend analysis
- Seasonal demand patterns

## Outcome
Insights from this analysis help hotels understand demand drivers and reduce cancellations through better pricing and policy decisions.
